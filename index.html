Media Queries

Media Queries are useful when you want to modify your site or app, check the width and height of the device, and the resolution.


Resumé

I would say that the Skeleton framework is best for programmers who are relatively new to the field. It's called, "dead simple" for a reason. Another pro would be it has superior main types and web layouts. Limited templates, and lack of CSS richness are some drawbacks for Skeleton. Compared to the Media Queries, I found Skeleton to be easier.

Flexbox

I’ve used Flexbox in IMM 120 and I’d say that it is a brilliant way of handling a fluid layout and coupled with media queries, it’s going to get the job done. I was never familiar with media queries and how to incorporate them in my websites, since I never had to make my websites compatible for all devices, but I must say that I am starting to get the hang of it now.


Bootstrap

Like Flexbox, I thought that Bootstrap was fairly simple. Both are clean, organized grids. Great standardized platform with all the basic styles and components needed (layout grid, panels, tables, buttons, modals, form styles, etc) to build on which saves a lot of time.
